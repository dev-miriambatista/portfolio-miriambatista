Login Responsivo

Rede Social
  • Abrir site
  • GitHub

Este código HTML e CSS cria uma página de rede social. A estrutura HTML contém seções para exibir um telefone simulado com uma tela de navegação e ícones de redes sociais. O CSS define estilos para o layout da página, incluindo a posição do telefone e dos ícones das redes sociais. O fundo da página é uma imagem de madeira e os ícones têm efeitos de sombra e transição quando passados por cima.

Parallax

  • GitHub

Este código HTML e CSS cria uma página de projeto de cordel moderno. A estrutura HTML contém seções para diferentes partes do poema, incluindo texto e imagens. O CSS define estilos para o layout da página, como as fontes, cores de fundo e sombras. O cabeçalho contém o título do cordel e informações sobre o autor. Este código HTML e CSS estrutura o conteúdo da página com cabeçalho, navegação, conteúdo principal e rodapé. O CSS define os estilos visuais, como cores, fontes e layout da página.
